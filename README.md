# insertionSort
O algoritmo de ordenação Insertion Sort é um dos algoritmos de ordenação mais simples e eficazes. Ele classifica um conjunto de elementos, como um array, construindo uma sequência ordenada um elemento de cada vez. O algoritmo é chamado de "Insertion Sort" porque ele insere cada elemento não ordenado na posição correta na sequência ordenada. O processo é semelhante a organizar cartas em uma mão, uma carta de cada vez.

Aqui está uma descrição do algoritmo Insertion Sort:

O algoritmo começa considerando o primeiro elemento do array como uma sequência ordenada, pois, no início, só há um elemento.

Em seguida, ele passa para o segundo elemento e o compara com o primeiro. Se o segundo elemento for menor, eles trocam de posição, colocando o menor no lugar do primeiro elemento.

O algoritmo continua avançando, comparando e trocando elementos até que todos os elementos do array tenham sido percorridos.

Para cada elemento subsequente, o algoritmo verifica se o elemento atual é menor do que o elemento à sua esquerda. Se for, ele continua a retroceder na sequência ordenada, movendo elementos para a direita até encontrar a posição correta para o elemento atual.

O algoritmo repete o processo até que todos os elementos estejam em sua posição correta na sequência ordenada.

O Insertion Sort é eficaz para conjuntos de dados pequenos, mas sua complexidade de tempo é quadrática, o que significa que seu desempenho pode ser lento em grandes conjuntos de dados. No entanto, é um algoritmo estável, o que significa que ele mantém a ordem relativa dos elementos com chaves iguais e não requer espaço de memória adicional, tornando-o uma escolha apropriada em alguns cenários.
